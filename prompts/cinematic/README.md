## Contributing

See the main [CONTRIBUTING.md](../../CONTRIBUTING.md) for guidelines on submitting cinematic prompts.

**Cinematic Category Requirements**:
- Must feature character(s) with narrative context
- Traditional cinematography techniques (tracking shots, composition rules)
- Emotional or dramatic impact
- Clear story moment or scene

**Good Examples**:
- Character-driven scenes with specific emotions
- Establishing shots that set narrative tone
- Dramatic moments with lighting and framing

**Avoid**:
- Static product shots without narrative
- Abstract visuals without character focus
- Generic "beautiful" scenes without story context
